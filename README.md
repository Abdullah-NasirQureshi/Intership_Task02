# Smart Inventory & Sales Management System (SISMS)

A MERN stack application for managing inventory, sales, and customers for small and medium-sized businesses.

## Team: Omicron

## Project Duration
17th Jan to 16th Feb, 2026

## Technology Stack
- **Frontend**: React.js, Redux Toolkit, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Deployment**: Vercel (Frontend), Render (Backend), MongoDB Atlas

## User Roles
- **Admin**: Full system access, user management, analytics
- **Staff**: Manage products, stock, customers, and sales invoices

## Getting Started

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Configure your environment variables
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

## Features
- Secure JWT authentication
- Product and category management
- Stock in/out tracking
- Sales invoice generation
- Customer management
- Reports and dashboards
